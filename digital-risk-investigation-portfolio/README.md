# Digital Risk Investigation Portfolio

## Overview

This is a self-learning mini project focused on phishing URL investigation, suspicious domain review, and digital risk case documentation.

The goal of this project is to practice entry-level Digital Risk Investigator skills, including public source validation, URL analysis, reputation checking, evidence collection, threat classification, and structured English reporting.

## Tools Used

- PhishTank
- VirusTotal
- Google Search
- WHOIS Lookup
- DNS Lookup
- Google Sheets / CSV
- Markdown

## Skills Demonstrated

- Phishing URL investigation
- Suspicious URL and domain checking
- Public threat intelligence source validation
- Evidence collection
- Threat classification
- Structured investigation report writing
- Safe URL documentation using defanged URLs

## Safety Note

All suspicious URLs are defanged for safety. No credentials were entered, no files were downloaded, and no direct interaction was performed with phishing websites. This project is for educational and portfolio purposes only.
